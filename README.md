# 🎮 Simon Game

A web-based implementation of the classic **Simon Game**, built with **HTML, CSS, JavaScript, and jQuery**.  
The game challenges players to remember and repeat an ever-growing sequence of colors and sounds.

---

## 🚀 Features
- Random sequence generation each round
- Interactive buttons with **flash + sound effects**
- User input tracking and validation
- Game-over detection with restart option
- Responsive design for different screen sizes

---

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling and animations
- **JavaScript (ES6)** for game logic
- **jQuery** for DOM manipulation and event handling
- **Audio API** for sound effects

---

## 🎯 How to Play
1. Press any key to start the game.
2. Watch the sequence of flashes and sounds.
3. Repeat the sequence by clicking the colored buttons.
4. Each round adds one more step to the sequence.
5. If you make a mistake, the game ends -> Press any key to restart.
